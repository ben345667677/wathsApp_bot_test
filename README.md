# WhatsApp Bot - בוט וואטסאפ

בוט WhatsApp מבוסס whatsapp-web.js שמאזין להודעות בקבוצה ומחזיר תגובות אוטומטיות.

## 📋 דרישות

- Node.js (גרסה 14 ומעלה)
- חשבון WhatsApp
- חיבור אינטרנט יציב

## 🚀 התקנה

1. התקן את התלויות:
```bash
npm install
```

## ⚙️ הגדרה

ערוך את הקובץ `bot.js` ושנה את ההגדרות הבאות:

```javascript
const CONFIG = {
    // שנה את שם הקבוצה שלך כאן
    TARGET_GROUP_NAME: 'שם הקבוצה שלך',

    // הוסף/ערוך תגובות
    RESPONSES: {
        'שלום': 'שלום! איך אפשר לעזור?',
        // הוסף תגובות נוספות כאן...
    }
};
```

## ▶️ הפעלה

הפעל את הבוט:
```bash
npm start
```

או:
```bash
node bot.js
```

## 📱 חיבור הבוט

1. הרץ את הבוט
2. יופיע קוד QR במסך
3. פתח WhatsApp בטלפון
4. לחץ על שלוש הנקודות > מכשירים מקושרים
5. לחץ על "קשר מכשיר"
6. סרוק את קוד ה-QR

## 🎯 פקודות זמינות

- `!עזרה` - הצגת רשימת פקודות
- `!מידע` - מידע על הקבוצה
- `!זמן` - השעה הנוכחית
- `!כולם` - קריאה לכל המשתתפים
- `!מזג אויר` - מזג האוויר

### מילות מפתח אוטומטיות:
- שלום, היי, תודה, בוט, עזרה

## 🔧 התאמה אישית

### הוספת תגובות חדשות:

```javascript
RESPONSES: {
    'מילת מפתח': 'התגובה שתחזור',
    'שאלה': 'תשובה',
}
```

### הוספת פקודות חדשות:

ערוך את הפונקציה `handleCommand()` והוסף:

```javascript
case '!הפקודה שלי':
    await message.reply('התגובה שלי');
    break;
```

## 🛡️ אבטחה

- הבוט שומר את פרטי האימות באופן מקומי בתיקייה `.wwebjs_auth`
- לא תצטרך לסרוק QR בכל פעם שהבוט מופעל
- אל תשתף את תיקיית האימות עם אחרים

## ⚠️ הערות חשובות

1. **קבוצה אחת**: הבוט מוגדר להאזין לקבוצה אחת. שנה את `TARGET_GROUP_NAME`.
2. **פרטיות**: הבוט יכול לראות את כל ההודעות בקבוצה.
3. **מגבלות**: WhatsApp עשויה לחסום בוטים שפעילים מדי.
4. **ספאם**: אל תשלח יותר מדי הודעות בזמן קצר.
